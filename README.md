# strangesentencegenerator.js
変な文を作る JavaScript のライブラリです。

## 目次
- ssg.js  
  ライブラリ本体です
- ssg.min.js  
  ライブラリ本体のです。圧縮版

## 説明
- ユニコードの日本語部の記号、アラビア文字、ジャワ文字、錬金術記号をランダムに組み合わせた文字列を生成します
- 前項のどの文字を使うのか選択することができます
